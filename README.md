Personal Portfolio
A modern and responsive personal portfolio website built using React.js and pnpm. This portfolio showcases my skills, projects, and contact information in a clean and user-friendly interface.

🧩 Components
The application is structured into the following components:

Home: Introduction and hero section.

About: Information about me and my background.

Skills: A list of my technical skills.

Projects: Showcase of my projects with descriptions and links.

Testimonials: Feedback from colleagues and clients.

Contact: Contact form and social media links.

Footer: Footer with copyright and additional links.

Loader: A loading spinner displayed during the initial load.

Navbar: Navigation bar for easy access to different sections.

⚙️ Technologies Used
React.js: JavaScript library for building user interfaces.

pnpm: Fast, disk space efficient package manager.

React Router: Declarative routing for React.js.

CSS: Styling the components.

React Hooks: Managing state and lifecycle methods.

🚀 Getting Started
Prerequisites
Ensure you have the following installed:

Node.js (v14 or higher)

pnpm (v6 or higher)
